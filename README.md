# demo-aplicacion-dashboard

Repositorio de demostración para una aplicación multiplataforma.

## Estructura

- `mobile/`: aplicación Expo/React Native con login, chat y reporte de incidente.
- `web/`: carpeta reservada para el panel de control web.
- `backend/`: carpeta reservada para la lógica del backend.
- `shared/`: carpeta reservada para código compartido entre plataformas.

Cada subcarpeta tiene su propio `README.md` con detalles de lo que contiene o lo que está planeado.
