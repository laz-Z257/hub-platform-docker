# Resumen de Sesión - 2026-07-16

## Problemas Resueltos

### 1. Expo Symbols Error (splash screen)
**Problema:** La app se quedaba colgada en la pantalla de carga.
**Causa:** Faltaba el archivo `symbols.json` en `expo-symbols`.
**Solución:** Se actualizó el `postinstall` en `mobile/package.json` para copiar también `symbols.json`.

### 2. Splash Screen Nunca se Ocultaba
**Problema:** El splash screen nunca se ocultaba, la app quedaba en negro.
**Causa:** `SplashScreen.hideAsync()` nunca se llamaba.
**Solución:** Se agregó `useEffect` en `_layout.tsx` para ocultar el splash al cargar.

### 3. Connectivity Context Muy Agresivo
**Problema:** Mensaje de "sin internet" aparecía por una falla temporal.
**Solución:** Se cambió a requerir 3 fallos consecutivos antes de mostrar offline.

### 4. Base de Datos - Migraciones Faltantes
**Problema:** El login/registro daba "Error interno del servidor" en producción.
**Causa:** Las migraciones 0011-0014 no existían en la BD de producción.
**Solución:**
- Se crearon las migraciones SQL faltantes
- Se modificó `docker-entrypoint.sh` para ejecutar migraciones al iniciar
- Se actualizó `schema.ts` con los índices
