# Mobile App

Este subproyecto contiene la aplicación móvil Expo/React Native.

## Qué incluye

- Pantalla de login
- Chat de soporte con mensajes simulados
- Pantalla de reporte de incidente
- Rutas con `expo-router`
- Estilos con `nativewind`

## Comandos

```bash
cd mobile
npm install
npm start
```

Para ejecutar en Android, iOS o web usa:

```bash
npm run android
npm run ios
npm run web
```

## Notas

La lógica actual del login, chat y reporte es de demostración local. El backend aún no está implementado.
